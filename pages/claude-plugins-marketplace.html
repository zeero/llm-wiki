claude-plugins.dev (Marketplace & CLI Plugin Manager)

Claude Code のプラグインと Agent Skills を検索・インストールできる、コミュニティ運営の registry と CLI。

概要

claude-plugins.dev は、Claude Code のプラグイン（skills を含む）を発見・管理するためのコミュニティ運営の registry と CLI（claude-plugins）。Claude Code は通常 marketplace 追加とプラグインインストールの 2 ステップを要するが、この CLI は両方を 1 コマンドで自動化する。OSS（Kamalnrf/claude-plugins）で、bun・Val Town・Claude Code で構築されている。Claude だけでなく Cursor、VS Code、OpenCode、Codex、Amp Code、Letta、Goose とも互換とうたう。

詳細

CLI コマンド

npx claude-plugins install <unique-plugin-identifier>   # インストール
npx claude-plugins list                                  # 一覧
npx claude-plugins enable <plugin-name>                  # 有効化
npx claude-plugins disable <plugin-name>                 # 無効化

Claude Code v2.0.12 以上が必要。

掲載プラグインの例

registry には Anthropic 公式・コミュニティ製の多数のプラグインが並ぶ。ダウンロード数・スター数でソート可能で、代表例:

  • frontend-design（@anthropics/claude-code-plugins）: 汎用 AI っぽさを避けた高品質フロントエンド生成
  • document-skills（@anthropics/anthropic-agent-skills）: Excel / Word / PowerPoint / PDF 処理スイート
  • compound-engineering（@EveryInc）: 29 agents・22 commands・19 skills を含む複合開発ツール
  • code-review / pr-review-toolkit / feature-dev（Anthropic）: レビュー・機能開発ワークフロー
  • context7（@anthropics/claude-plugins-official）: 最新ドキュメント参照用 MCP サーバー
  • @wshobson/claude-code-workflows 系: python / backend / javascript-typescript / database-design など言語・領域別
