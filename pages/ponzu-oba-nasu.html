旨だれやみつき大葉なす

焼いたなすを、味ぽん（ぽん酢）ベースの旨だれに大葉と一緒に漬ける、なす大量消費レシピ。ミツカン公式「おうちレシピ」より。麺つゆ漬けの新定番。

材料・手順（要点）

材料（4人分）

  • なす … 3本
  • 大葉 … 10枚
  • ごま油 … 適量

【A】ぽん酢（味ぽん）100ml / 水 50ml / 砂糖 大さじ1 / おろしにんにく 小さじ1/2 / おろししょうが 小さじ1/2 / いりごま（白）大さじ1 / ごま油 大さじ1 / ラー油 適量

手順

  1. なすは縦に1cm幅に切る。大葉は茎を取り除く。
  2. フライパンにごま油を中火で熱し、なすの両面に軽く焼き色がつくまで焼く（しんなりでOK）。
  3. 耐熱容器に【A】を入れて軽く混ぜ、ふんわりラップをして電子レンジ（600W）で約2分加熱する（突沸に注意）。
  4. 温めた漬けだれになすと大葉を加えてさっと混ぜ、冷蔵庫で5分ほど冷やして完成。

ぽん酢の酸味に、にんにく・しょうが・ごま油のコク、ラー油の辛みが効いてさっぱりコク深い。残った旨だれは冷しゃぶサラダなどに再利用可。
