swiftui-expert-skill

AvdLee/SwiftUI-Agent-Skill リポジトリに含まれる、実際にインストールして使う SwiftUI ベストプラクティスのスキル本体。

概要

swiftui-expert-skill は、SwiftUI Agent Skill（AvdLee/SwiftUI-Agent-Skill）リポジトリが提供するスキルの実体。AI コーディングツールに modern SwiftUI の知識（state management、非推奨 API の置き換え、パフォーマンス、iOS 26+ Liquid Glass 対応など）を与える。導入時に指定するスキル名でもある。

詳細

導入

skills.sh 経由で 1 コマンドで追加でき、--skill にこのスキル名を指定する。

npx skills add https://github.com/avdlee/swiftui-agent-skill --skill swiftui-expert-skill

構造

swiftui-expert-skill/SKILL.md を本体とし、references/ 配下に image-optimization / layout-best-practices / liquid-glass / list-patterns / modern-apis / performance-patterns / scroll-patterns / sheet-navigation-patterns / state-management / text-formatting / view-structure の各リファレンスを持つ。詳細な機能・背景は親ページ SwiftUI Agent Skill を参照。
