ハナレグミ Live News RSS化 開発計画

ハナレグミ公式サイトのライブニュース一覧をスクレイピングして RSS フィード化する開発計画（Claude Artifact）の要約。

概要

ハナレグミ公式サイトの Live ニュース一覧（https://www.hanaregumi.jp/news_category/live）をスクレイピングし、RSS フィードとして配信する開発計画。GitHub Actions で毎日12時（JST）に自動実行し、アーティファクトとして出力する。1ページ目のみ取得（25件、ページネーション対象外）。

詳細

技術スタック

  • HTML取得: requests
  • HTMLパース: beautifulsoup4
  • RSS生成: feedgen

JavaScript レンダリングなしで普通にHTMLが取得できるため Playwright は不要。

スクリプト構成と処理フロー

scripts/scrape.py にワンスクリプトで実装:

  1. requests でHTML取得
  2. beautifulsoup4 でパースし日付・タイトル・URLを抽出
  3. feedgen でRSS生成
  4. output/hanaregumi-live.xml へ出力

GitHub Actions

cron: "0 3 * * *"（JST 12:00 = UTC 03:00）と workflow_dispatch（手動実行）でトリガー。Python 3.12 セットアップ後に依存をインストールしてスクレイピングを実行し、成果物を actions/upload-artifact でアップロード。

注意事項

  • CRON は UTC 基準（JST 12:00 = UTC 03:00）。
  • アーティファクトは既存ワークフロー/リポジトリに統合する。
  • セレクタはローカル確認時に実際の DOM 構造に合わせて調整が必要。
