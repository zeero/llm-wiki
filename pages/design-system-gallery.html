Design System Gallery

54の著名Webサイトのデザインシステムを markdown + HTMLプレビューで集めたキュレーションギャラリーの要約。

概要

Design System Gallery は、Awesome Design MD（VoltAgent）を基に構築された、54の人気Webサイトのデザインシステム markdown と HTML プレビューのコレクション。各アイテムはサイト名とカテゴリ（AI, SaaS, Developer, Fintech, Design 等）でタグ付けされている。ライト/ダークの表示切替に対応。

詳細

デザインシステムの markdown 化は、AIコーディングツールに各ブランドのデザインガイドライン（色・タイポグラフィ・コンポーネント原則等）を読み込ませる用途に向く。

収録されている主なサイト（54件）:

  • AI: Claude, Cohere, ElevenLabs, Lovable, MiniMax, Mistral AI, Ollama, Replicate, Runway, Together AI, xAI
  • Developer: Composio, Cursor, Expo, OpenCode, Resend, Sentry, Vercel, VoltAgent, Warp
  • Fintech / Crypto: Coinbase, Kraken, Revolut, Stripe, Wise
  • Productivity: Linear, Notion, Raycast, Superhuman
  • Database: ClickHouse, MongoDB, Supabase
  • Design: Figma, Framer, Webflow
  • その他: Airbnb（Marketplace）, Apple（Consumer）, BMW（Automotive）, Cal.com, Clay, IBM, Intercom, Miro, Mintlify, NVIDIA, Pinterest, PostHog, Sanity, SpaceX, Spotify, Uber, Zapier, HashiCorp, Airtable
