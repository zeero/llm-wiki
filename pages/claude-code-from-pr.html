Claude Code --from-pr（PRセッション再開）

Claude Code v2.1.27（2026年1月30日リリース）で追加された --from-pr フラグと、セッション自動PRリンク機能を解説した記事（NaokiIshimura）の要約。

概要

--from-pr は PR番号または PR URL を指定するだけで、PRに紐付いた過去のセッションを再開 できるCLIフラグ。従来の --continue / --resume はセッションID・セッション名での管理だったため「あのPRの作業はどのセッションだったか」を探す手間があったが、--from-pr では開発者に最も直感的な識別子（PR番号）で会話履歴・ツール状態・コンテキストを完全復元できる。

詳細

使い方

claude --from-pr 42
claude --from-pr https://github.com/owner/repo/pull/42

実行すると、指定PRに紐付いたセッションを検索し、見つかれば --resume と同じ仕組みでそのセッションを再開する。

セッション自動PRリンク

v2.1.27 のもう一つの新機能。Claude Code セッション内で gh pr create を実行してPRを作成すると、そのセッションが自動的にPRにリンクされる。手動設定は不要。

前提条件

  • GitHub CLI（gh）がインストール済みで gh auth login で認証済み
  • gh pr create 等でセッションがPRにリンクされていること

セッション再開コマンドの整理

  • --continue（-c）: 直近のセッションを継続（識別子なし）
  • --resume（-r）: セッションID / 名でピッカーから再開
  • --from-pr: PR番号 / URL でPR紐付きセッションを再開

PRレビューステータスとの併用

v2.1.20 で追加されたPRレビューステータスインジケーター（🟢 Approved / 🟡 Pending / 🔴 Changes req. / 🟣 Merged / ⚪ Draft）がフッターに表示され、60秒ごとに自動更新。複数PR並行作業でもPRごとにセッションが分かれるためコンテキストが混在しない。
