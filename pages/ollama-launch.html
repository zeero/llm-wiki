Methods

ollama launch

Ollama v0.15 で追加された、AI コーディングツールの実行環境を1コマンドで立ち上げるコマンド。

概要

ollama launch claude-code を実行するだけで、環境変数・設定ファイル・セットアップ手順なしに Claude Code をローカル/クラウドの Ollama モデルで起動できる、という Ollama v0.15 の新機能を紹介する記事。従来は Ollama インストール → モデル pull → 環境変数設定（ANTHROPIC_AUTH_TOKEN / ANTHROPIC_BASE_URL）→ Claude Code インストール → 起動、と多段階だったものが1行になる。

記事は promotional なトーンで、数値やベンチマークは著者提示のもの。事実確認は Ollama 公式を優先すること。

詳細

対応ツールと使い方

ollama launch claude-code   # Anthropic 公式 CLI
ollama launch opencode      # OSS 代替
ollama launch codex         # OpenAI 互換
ollama launch droid         # モバイル開発特化

モデル

  • ローカル（GPU あり）: glm-4.7-flash（23GB+, 128K）、qwen3-coder（16GB+, 64K）、gpt-oss:20b。
  • クラウド（GPU なし/巨大モデル）: gpt-oss:120b-cloud、qwen3-coder:480b-cloud、glm-4.7:cloud。--model で指定（例 ollama launch claude-code --model qwen3-coder:480b-cloud）。

料金（記事の記載）

無料枠はセッション5時間/日でクラウドモデル利用可・月額$0。有料は Pro $20/月（無制限）、Team $50/月/人。記事は「Claude API $100+/月 に対し Ollama Pro $20/月」とコスト削減を強調している。

導入手順

ollama update                                   # 既存ユーザー
curl -fsSL https://ollama.com/install.sh | bash # 新規
ollama launch claude-code
