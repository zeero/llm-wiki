Claude Code

Anthropic が提供する、ターミナルで動作するエージェント型コーディングツール。Claude モデルを使い、コードベースの調査・編集・コマンド実行を自律的に行う。

概要

Claude Code は、開発者のターミナル上で動く AI コーディングエージェント。自然言語の指示を受けて、リポジトリ内のファイル探索・コード編集・テスト実行・Git 操作などを一連の作業として自律実行する。単なる補完ではなく「エージェントとして手を動かす」点が特徴で、この Vault にも Claude Code 関連の知見が多数集約されている。

詳細

拡張・制御の仕組み

Claude Code は挙動をカスタマイズする複数のレイヤーを持つ。

  • CLAUDE.md: プロジェクト／ユーザーごとの常時コンテキスト（指示・規約）。
  • Slash Commands / Skills: 定型ワークフローや手順書を呼び出す仕組み（custom slash commands は skills に統合されている）。
  • Subagents（サブエージェント）: 特化タスクを別コンテキストに切り出して委譲。
  • Hooks: ツール使用前後などのイベントでスクリプトを実行。
  • MCP（Model Context Protocol (MCP) Servers）: 外部ツール・データ源を接続する標準プロトコル。
  • Plugins / Marketplaces / Permissions / Settings: 配布・権限・設定管理。

エコシステムでの位置づけ

Claude Code はマルチエージェント基盤の中核としても扱われ、Multi-Agent Orchestration で紹介される claude-flow のように「Claude Code 特化」で多数の専門エージェントを編成するフレームワークも登場している。周辺ツールやベストプラクティスは Claude Code 強化ツール集・claude-code-best-practice リポジトリ に整理されている。
