Model Context Protocol (MCP) Servers

LLM にツールやデータソースへの安全なアクセスを与えるオープンプロトコル MCP と、その公式リポジトリ modelcontextprotocol/servers。

概要

Model Context Protocol（MCP） は、Large Language Model に対してツールやデータソースへのセキュアで制御された access を与えるためのオープンプロトコル。GitHub の modelcontextprotocol/servers リポジトリ（MIT ライセンス）は、その reference implementation 群と、コミュニティ製サーバー・追加リソースへの参照をまとめたカタログ。各 MCP サーバーは通常いずれかの MCP SDK で実装される。

詳細

公式 SDK

C# / Go / Java / Kotlin / PHP / Python / Ruby / Rust / Swift / TypeScript の各 MCP SDK が提供されている。

Reference servers

MCP の機能と公式 SDK をデモする参照サーバー:

  • Everything: prompts・resources・tools を備えたリファレンス/テスト用サーバー
  • Fetch: Web コンテンツの取得と LLM 向け変換
  • Filesystem: アクセス制御付きの安全なファイル操作
  • Git: Git リポジトリの読み取り・検索・操作
  • Memory: knowledge graph ベースの永続メモリ
  • Sequential Thinking: 思考シーケンスによる動的・省察的な問題解決
  • Time: 時刻・タイムゾーン変換

GitHub / GitLab / Google Drive / Google Maps / PostgreSQL / Puppeteer / Redis / Slack / SQLite などは archived となり servers-archived へ移動。

Official integrations

各社が自社プラットフォーム向けに本番品質の MCP サーバーを提供している。膨大な数が掲載され、例として 21st.dev Magic（UI コンポーネント生成）、ActionKit by Paragon（130+ SaaS 連携）、AWS、Azure、Atlassian（Jira / Confluence）、Alpaca（株式・オプション取引）、AlphaVantage（金融データ）、Apify、Algolia などがある。
