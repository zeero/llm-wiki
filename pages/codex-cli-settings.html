Codex CLI の機能・設定

OpenAI の Codex CLI を使いこなすための設定ファイル・便利機能・主要コマンドのまとめ。

概要

Codex CLI は 2025年4月に公開された OpenAI の CLI エージェント。短サイクルで改善が続き、GPT-5 公開（2025年8月7日）以降は ChatGPT 有料プラン（Plus/Pro/Team）で追加料金なしに利用できるようになった。公式ドキュメントはなくリポジトリを直接見る必要がある。Claude Code の性能劣化の声を背景に移行を検討する開発者が増えているという文脈で、実用的な設定を紹介する。

詳細

基本設定

  • インストール: npm install -g @openai/codex または brew install codex。
  • AGENTS.md: ~/.codex/AGENTS.md に共通プロンプトを書くと全会話に自動読み込み。日本語応答をデフォルトにするなら「日本語で簡潔かつ丁寧に回答してください」等。AGENTS.md は Cursor・GitHub Copilot・Gemini CLI・Roo Code など多くのツール共通の Custom Instructions ファイル名。
  • 設定ファイル ~/.codex/config.toml: Rust で再実装された経緯から TOML 形式。--config key=value（CLI 引数）> 環境変数 > config.toml の順で上書きされる。

便利な機能・設定

  • 推論を常に high に: model_reasoning_effort = "high"（既定は medium）。記事内で最もおすすめとされる設定。
  • Notify（Claude Hooks 相当）: notify = ["bash", "-lc", "afplay /System/Library/Sounds/Ping.aiff"] でタスク完了時に音。
  • MCP: 専用 JSON ではなく config.toml に [mcp_servers.<name>] として記述。現時点では stdio 通信のみ対応で、SSE のリモート MCP は mcp-remote / mcp-proxy などのラッパー経由（例: Figma）。
  • Web 検索: [tools] の web_search = true（デフォルト無効）。
  • Custom Prompts（スラッシュコマンド相当）: ~/.codex/prompts/ に置く（プロジェクト配下の .codex/prompts/ は読み込まれない）。
  • reasoning ログ非表示: hide_agent_reasoning = true（既定 false）。
  • @path/to/file: プロンプト内でファイル内容を明示指定。
  • VS Code 拡張（2025年8月27日公開）: 開いているファイルを参照でき @ 指定不要、Codex Cloud へタスクを流せる。

主要コマンド

コマンド Claude Code 相当 説明
/new /clear セッションを破棄して新規開始
/mcp /mcp 有効な MCP を表示
/compact /compact 対話履歴を要約しトークン節約

利用制限（記事時点の目安）

Plus/Business/Enterprise/Education のローカルタスクは週制限内で 5 時間あたり 30〜150 メッセージ、Pro は 300〜1,500 メッセージ（p25〜p75 の目安、今後変更されうる）。
