Command R+

Cohere の多言語対応大規模言語モデル。日本語応答、長いコンテキスト、RAG、Tool Use、ローカル実行の紹介。

概要

2024年のクリップでは、Command R+ を GPT-4 と同等の性能を持つモデルとして紹介している。記事中の仕様は 1,040億パラメーター、12万8,000トークンのコンテキストウィンドウで、日本語を含む多言語処理に対応するというものだった。

特徴と実行方法

記事の比較では、Multilingual と RAG で Mistral Large を上回り、Tool Use で GPT-4 Turbo を超える結果が紹介されている。また、Hugging Face の CohereForAI/c4ai-command-r-plus から取得でき、Web 上のチャットデモも案内されている。

これらは 2024年時点の紹介・比較であり、現在のモデル提供状況やベンチマークを表すものではない。ローカル実行時はモデルサイズ、量子化、利用可能なメモリ、ライセンスを別途確認する必要がある。
