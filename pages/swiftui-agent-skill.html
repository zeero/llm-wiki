SwiftUI Agent Skill

Antoine van der Lee（AvdLee）らが公開した、AI コーディングツール向けの SwiftUI ベストプラクティス Agent Skill。

概要

SwiftUI-Agent-Skill は、Agent Skills open format に準拠した OSS（MIT ライセンス）で、あらゆる AI コーディングツールに実践的な SwiftUI ガイダンスを与えるスキル。modern SwiftUI API、state management、performance、iOS 26+ の Liquid Glass 対応などの知識を、エージェントやコードレビュー向けに簡潔なリファレンスへ蒸留している。作者は SwiftLee ブログで知られる Antoine van der Lee と Omar Elsayed。

詳細

できること

  • state management ツールの選択（@State / @Binding / @Observable / @Bindable）
  • 非推奨 SwiftUI API に対する modern な置き換えの提案
  • sheet、navigation、scrolling、list に関するガイダンス
  • iOS 26+ Liquid Glass の利用（availability fallback 付き）
  • view identity の安定化（ForEach の落とし穴回避）、view composition の改善
  • 不要な re-render を招く update/refresh の落とし穴回避
  • パフォーマンス改善（hot path での state 更新削減、list 性能、UIImage(data:) の downsampling 提案）

設計方針は Non-Opinionated（アーキテクチャや project 構造を強制しない）・Modern-first・Practical & concise。

インストール

skills.sh 経由で 1 コマンド:

npx skills add https://github.com/avdlee/swiftui-agent-skill --skill swiftui-expert-skill

Claude Code では marketplace 追加（/plugin marketplace add AvdLee/SwiftUI-Agent-Skill）→ プラグインインストール、または .claude/settings.json の enabledPlugins / extraKnownMarketplaces でプロジェクト全体に配布できる。Codex や Cursor など他ツールでも各公式手順で導入可能。

スキル構造

swiftui-expert-skill/SKILL.md に加え、references/ 配下に image-optimization / layout-best-practices / liquid-glass / list-patterns / modern-apis / performance-patterns / scroll-patterns / sheet-navigation-patterns / state-management / text-formatting / view-structure の各リファレンスを持つ。関連スキルとして Swift Concurrency Expert、Core Data Expert がある。
