MCP Apps

MCP 初の公式拡張機能「MCP Apps」——AIがチャット内でインタラクティブなUIを表示できる仕組みを紹介した記事（emi_ndk, 2026年1月）の要約。

概要

MCP Apps は Model Context Protocol の初の公式拡張機能。2026年1月26日にMCP公式ブログで発表され、「ツールがインタラクティブなUIコンポーネントを会話内で直接レンダリングできる」ようになった。AIとの対話がテキストのやり取りから、ダッシュボード・フォーム・グラフ・リアルタイムモニタリングなどのUI操作へ進化する。

詳細

技術的な仕組み

2つの重要なプリミティブで構成される:

  • UIメタデータ付きツール: ツールが _meta.ui.resourceUri でUIリソースへの参照を返す。
  • UIリソース: ui:// スキームで提供されるサーバーサイドリソース。HTML/JavaScript がバンドルされている。

ホスト（Claude、ChatGPT など）がリソースを取得し、サンドボックス化された iframe 内でレンダリング。JSON-RPC と postMessage で双方向通信する。

用途

データ探索ダッシュボード（ドリルダウン・フィルタ）、依存フィールドを持つ設定ウィザード、ドキュメントレビュー（PDFインライン表示）、リアルタイムモニタリングなど。

開発者SDK

@modelcontextprotocol/ext-apps パッケージを提供。app.onToolResult() でツール結果を受け取り、app.callTool() でUIからサーバーツールを呼び、app.updateContext() でモデルコンテキストを更新する。フレームワーク非依存の標準 postMessage 通信のため React / Vue / Svelte など任意のものが使える。

対応クライアントとセキュリティ

Claude（Web/Desktop）、VS Code Insiders、Goose が対応済み。ChatGPT もロールアウト中。セキュリティは iframe サンドボックス、事前宣言テンプレート（ホストが事前レビュー可能）、監査可能な JSON-RPC メッセージ、ツール実行前のユーザー同意で担保。
