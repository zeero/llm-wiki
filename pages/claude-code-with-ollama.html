Claude Code with Ollama（Anthropic API 互換）

Ollama の Anthropic Messages API 互換対応により、Claude Code をローカル/クラウドのオープンモデルで動かす方法をまとめた Ollama 公式ドキュメント・ブログの要約。

概要

Ollama v0.14.0 以降は Anthropic の Messages API と互換になり、Claude Code（Anthropic のターミナル型エージェント開発ツール）を glm-4.7 / qwen3-coder / gpt-oss などのオープンモデルで利用できるようになった。ローカルモデルでもクラウドモデル（ollama.com 経由）でも動作する。

詳細

セットアップ

Claude Code をインストール後、環境変数で Ollama を指すよう設定する:

export ANTHROPIC_AUTH_TOKEN=ollama
export ANTHROPIC_BASE_URL=http://localhost:11434
claude --model gpt-oss:20b

クラウドモデルも利用可能（例: claude --model glm-4.7:cloud）。Claude Code は大きなコンテキストウィンドウを要するため、少なくとも32K〜64Kトークンを推奨。Ollama のクラウドモデルは常にフルコンテキスト長で動作する。

推奨モデル

  • ローカル: gpt-oss:20b, qwen3-coder, glm-4.7, gpt-oss:120b
  • クラウド: glm-4.7:cloud, minimax-m2.1:cloud

Anthropic SDK 互換

既存の Anthropic SDK アプリは base URL を http://localhost:11434、api_key を ollama（必須だが無視される）に変えるだけで Ollama に接続できる。Python / JavaScript SDK に対応。

サポート機能

Messages とマルチターン会話、ストリーミング、システムプロンプト、Tool calling / function calling、Extended thinking、Vision（画像入力）。
