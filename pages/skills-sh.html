skills.sh

AIエージェント用スキルのオープンエコシステム「skills.sh」（Vercel）と、人気スキルを紹介した記事（emi_ndk, 2026年1月）の要約。

概要

skills.sh は2026年1月に Vercel がローンチした、AIコーディングアシスタント用スキルの オープンエコシステム（”AIエージェント版 npm”）。ローンチ6時間で2万インストールを突破。npx skills add <owner/repo> の1コマンドで、Claude Code / Cursor / GitHub Copilot / Gemini / Windsurf / Cline / OpenAI Codex など15以上のAIコーディングツールにスキルをインストールできる。公開スキルは35,000以上。

詳細

主要スキル（記事時点のランキング）

  • vercel-react-best-practices（72.5K+）: Vercel公式の React/Next.js 最適化ノウハウ。40超のルールを8カテゴリに分類し CRITICAL〜LOW の影響度ラベル付き（例: バレルファイル回避で200-800ms短縮）。
  • find-skills（55.9K+）: スキル検索。
  • web-design-guidelines（55K+）: 100超のUI/UXルールでコードレビュー（アクセシビリティ・パフォーマンス・UX・国際化）。
  • remotion-best-practices（52.2K+）: React で動画を作る Remotion の公式スキル。
  • 他: frontend-design, playwright-automation, prompt-engineering（Anthropic公式）, stripe-integration, react-native-guidelines, vercel-deploy-claimable。

インストール方法

  • npx: npx skills add vercel-labs/agent-skills（推奨）
  • Claude Code プラグイン経由: /plugin marketplace add anthropics/skills → /plugin install ...
  • 手動: .claude/skills/ または ~/.claude/skills/ に SKILL.md（frontmatter に name / description / triggers）を配置

Claude はタスクに基づき関連スキルを自動で呼び出す（手動選択不要）。

注意点

  • セキュリティ: 公式・有名リポジトリ（vercel-labs, anthropics 等）優先、スクリプト内容を実行前に確認、APIキーは環境変数で。
  • パフォーマンス: スキルは必要最小限に（入れすぎるとコンテキスト圧迫）、長いスキルは分割し SKILL.md をメニューとして機能させる、トリガーを適切に設定。
