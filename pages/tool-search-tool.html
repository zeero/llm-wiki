Methods

Tool Search Tool

MCP ツールの定義を遅延読み込み（loaded on demand）し、セッション開始時のコンテキスト消費を抑える Claude Code の仕組み。

概要

Tool Search Tool は、多数の MCP ツールを一度に読み込まず、必要になった時点で定義を取得する（遅延読み込み）ことで、セッション開始時のコンテキスト消費を実質ゼロに抑える機能。ツールを大量に接続してもコンテキストを圧迫せず、ツール選択の精度も向上するとされる。

詳細

有効化

~/.claude/settings.json の env に以下を設定して有効化する。

{ "env": { "ENABLE_TOOL_SEARCH": "true" } }

位置づけ

MCP（Model Context Protocol (MCP) Servers）で多くの外部ツールを接続すると、各ツールの定義がコンテキストを消費する。Tool Search Tool は、定義を「使うときだけ」ロードすることでこの負荷を切り離す。Claude Code 強化ツール集 の「周辺ワークフローツール」節で、AI エージェント開発の速度を落とさないための一手として紹介されている。
